# 📊 Análisis de Salarios en el Área de Datos

## 📌 Descripción del Proyecto
Este proyecto analiza los salarios en el sector de datos utilizando Python.  
El objetivo es entender cómo variables como el rol y el nivel de experiencia impactan en los ingresos.

---

## 🧰 Herramientas utilizadas
- Python
- Pandas
- Seaborn
- Matplotlib

---

## 📂 Dataset
Se utilizó el dataset **Data Science Job Salaries**, que incluye:
- Títulos de trabajo
- Niveles de experiencia
- Salarios en USD
- Ubicación y tamaño de la empresa

---

## 🔍 Principales conclusiones

### 1. El crecimiento salarial no es lineal
Los salarios aumentan significativamente a partir de niveles senior y ejecutivos.

### 2. El rol define el potencial salarial
Roles como Data Engineer y Data Scientist presentan mayores ingresos que Data Analyst.

### 3. La experiencia impacta directamente en el salario
Se observa una progresión clara de ingresos dentro de cada rol a medida que aumenta la experiencia.

---

## 📊 Análisis realizados
- Limpieza y preparación de datos
- Análisis exploratorio (EDA)
- Salario por rol
- Salario por nivel de experiencia
- Análisis combinado (rol + experiencia)
- Visualizaciones
